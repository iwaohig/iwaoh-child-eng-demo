🎓 体験申込システム デモ

📝 システム概要

このデモは、既存のSalesforce + SPIRALシステムを、AWS + LINE WORKS WOFFに置き換える検証用モックアプリケーションです。

教室の体験申込受付から、教室の先生による申込管理までの一連の流れを体験できます。

📋 体験申込フォーム

お客様が体験申込を行うWebフォーム
（既存HTMLを改良）

👨‍🏫 管理画面 (WOFF)

教室の先生が申込を管理する画面
（LINE WORKS WOFF代替）

🏗️ システム構成

フロントエンド HTML/CSS/JavaScript
GitHub Pages
バックエンド AWS Lambda
API Gateway
データベース Amazon DynamoDB
(iwaoh-child-eng-*)
管理機能 LINE WORKS WOFF
(SPIRALの代替)

📊 機能仕様

  • 教室選択: 3つの教室（渋谷・新宿・池袋）から選択
  • 申込受付: 体験申込データをDynamoDBに保存
  • ステータス管理: 確認 → 予約未決定 → 予約済み → 保留中 → 結果決定
  • デモ用簡略化: 教室別分類なし（全データ表示）
  • AWS リソース: 全てのリソースに "iwaoh-child-eng-" プレフィックス

🔧 デプロイ手順

  1. GitHubリポジトリ作成: iwaoh-child-eng-demo
  2. GitHub Pagesでフロントエンドを公開
  3. AWS CloudFormationでバックエンドリソースを作成
  4. LINE WORKS WOFFアプリケーションを設定
